# Unified Fields

Use these canonical field names for every source. Unknown scalar values should be `null`; unknown array values should be `[]`.

| Field | Type | Notes |
| --- | --- | --- |
| `id` | string | Source-aware ID. |
| `source` | enum | `suumo`, `athome`, `homes`, `yahoo_japan`. |
| `source_listing_id` | string/null | Listing ID from the source site if visible. |
| `source_url` | string/null | Canonical detail page URL. |
| `source_site_name` | string/null | Human-readable source name. |
| `fetched_at` | string/null | ISO timestamp generated by the agent/task. |
| `listing_status` | string/null | Active, ended, sold, unknown, etc. |
| `title` | string/null | Listing title. |
| `property_type` | string/null | Site property type label. |
| `property_subtype` | string/null | More specific site category when available. |
| `transaction_type` | string/null | Sale, rent, investment, etc. |
| `price_jpy` | integer/null | Numeric yen amount where parseable. |
| `price_text` | string/null | Original price string. |
| `price_per_sqm_text` | string/null | Original per-square-meter price text. |
| `management_fee_jpy` | integer/null | Monthly management fee. |
| `management_fee_text` | string/null | Original management fee text. |
| `repair_reserve_fund_jpy` | integer/null | Monthly reserve fund. |
| `repair_reserve_fund_text` | string/null | Original repair reserve fund text. |
| `repair_reserve_fund_initial_text` | string/null | Repair reserve fund lump-sum/fund text. |
| `other_fee_text` | string/null | Other fees, expenses, utility, or miscellaneous cost text. |
| `deposit_key_money_text` | string/null | Rental deposits/key money if applicable. |
| `expected_annual_rent_text` | string/null | Expected annual rent for investment listings. |
| `gross_yield_text` | string/null | Yield text for investment listings. |
| `address` | string/null | Full listed address. |
| `prefecture` | string/null | Prefecture. |
| `city` | string/null | City. |
| `ward` | string/null | Ward. |
| `town` | string/null | Town/chome if available. |
| `latitude` / `longitude` | number/null | Only if shown or embedded clearly. |
| `nearest_stations` | array | Objects with `line`, `station`, `walk_minutes`, `bus_minutes`, `text`. |
| `building_name` | string/null | Building or mansion name. |
| `room_number` | string/null | Unit number if shown. |
| `layout` | string/null | `1LDK`, `2DK`, etc. |
| `floor_area_sqm` | number/null | Exclusive/internal area. |
| `floor_area_text` | string/null | Original floor area string, including measurement basis. |
| `land_area_sqm` | number/null | Land area. |
| `land_area_text` | string/null | Original land area string. |
| `balcony_area_sqm` | number/null | Balcony area. |
| `balcony_area_text` | string/null | Original balcony area string. |
| `building_area_sqm` | number/null | House/building area. |
| `building_area_text` | string/null | Original building area string. |
| `area_measurement_method` | string/null | Measurement basis such as wall-center, registered, etc. |
| `floor` | integer/null | Unit floor. |
| `total_floors` | integer/null | Above-ground floors. |
| `basement_floors` | integer/null | Basement floors. |
| `structure` | string/null | RC, SRC, 木造, etc. |
| `built_year` / `built_month` | integer/null | Completion date if parseable. |
| `age_text` | string/null | Original age label. |
| `total_units` | integer/null | Total units in building. |
| `units_for_sale` | integer/null | Number of units currently for sale in the listing. |
| `orientation` | string/null | Main direction. |
| `parking` | string/null | Parking availability/cost. |
| `rights` | string/null | Ownership/land rights. |
| `zoning` | string/null | Use district if available. |
| `current_status` | string/null | Vacant, occupied, leased, etc. |
| `building_status` | string/null | Source status such as used, new, unoccupied, etc. |
| `delivery_time` | string/null | Handover timing. |
| `management_form` | string/null | Building management form. |
| `management_staff` | string/null | Manager/staff status such as day-shift or patrol. |
| `management_company` | string/null | Management company if shown. |
| `construction_company` | string/null | Construction company/施工. |
| `defect_warranty` | string/null | Defect warranty label/value. |
| `defect_insurance` | string/null | Defect insurance label/value. |
| `certifications` | string/null | Evaluation/certification text. |
| `reform_renovation` | string/null | Reform/renovation status. |
| `energy_performance` | string/null | Energy performance text. |
| `insulation_performance` | string/null | Insulation performance text. |
| `estimated_utility_cost` | string/null | Estimated utility cost text. |
| `features` | array[string] | Normalized feature tags. |
| `equipment` | array[string] | Equipment/service tags. |
| `remarks` | string/null | Source remarks/notes. |
| `recommended_comment` | string/null | Agent/site recommendation text. |
| `sale_schedule` | string/null | Sale schedule text. |
| `event_info` | string/null | Event/open-house text. |
| `agency_name` | string/null | Listing agency. |
| `agency_phone` | string/null | Agency phone if shown. |
| `agency_address` | string/null | Agency address. |
| `agency_business_hours` | string/null | Agency business hours. |
| `agency_holidays` | string/null | Agency holidays/closed days. |
| `agency_license` | string/null | License number. |
| `agency_transaction_type` | string/null | Agency transaction role such as 仲介/媒介. |
| `contact_url` | string/null | Inquiry/contact URL. |
| `images` | array[string] | Listing images. |
| `floor_plan_images` | array[string] | Floor plan images. |
| `media` | array[object] | Media objects with `kind`, `url`, and `caption`. |
| `image_count` | integer/null | Source-reported or counted image count. |
| `detail_text` | string/null | Freeform description. |
| `source_published_at` | string/null | Source first-published date if shown. |
| `source_updated_at` | string/null | Source listing update date if shown. |
| `next_update_at` | string/null | Source next update date if shown. |
| `transaction_valid_until` | string/null | Transaction condition validity date. |
| `source_structured_fields` | array[object] | All original detail-page label/value pairs with section and normalized hint. |
| `search_metadata` | object | Applied filters, search path, and translation notes. |
| `raw` | object | Original labels and ambiguous fields. |
